Описание Архитектуры проекта

## Выбранная идея: Just Notes (Minimal Notes)

**Суть:** Расширение для Chrome, которое заменяет стандартную страницу новой вкладки на чистое поле для заметок с автосохранением.  
**Ценность:** Бесплатная, простая альтернатива платным расширениям (Momentum, New Tab Widgets) с единственной функцией — заметки на новой вкладке.  
**Целевая аудитория:** Пользователи, которым нужен быстрый доступ к заметкам без лишних виджетов, регистрации и подписки.

---

## Архитектура сервиса

Расширение полностью работает на клиентской стороне, без собственного бэкенда. Все данные хранятся локально в браузере пользователя.

### Компоненты:

1. **Manifest V3** — файл конфигурации, описывающий расширение, разрешения и точку входа.
2. **Override Page (`newtab`)** — HTML-страница, которая подменяет стандартную вкладку Chrome.
3. **Chrome Storage API** — для сохранения заметок (локально, без аккаунта).
4. **Скрипты** — JavaScript, обеспечивающий загрузку, автосохранение и (опционально) форматирование.

## Суть технической реализации

### 1. Файловая структура

```
just-notes-extension/
├── manifest.json          # конфигурация расширения
├── notes.html             # основная страница (новая вкладка)
├── styles.css             # стили (можно встроить в HTML)
├── script.js              # логика работы заметок
├── icon16.png             # иконка 16x16
├── icon48.png             # иконка 48x48
└── icon128.png            # иконка 128x128
```

### 2. Manifest V3

```json
{
  "manifest_version": 3,
  "name": "Just Notes",
  "version": "1.0.0",
  "description": "Replace new tab with a simple notepad. Auto-save, no clutter.",
  "chrome_url_overrides": {
    "newtab": "notes.html"
  },
  "permissions": ["storage"],
  "icons": {
    "16": "icon16.png",
    "48": "icon48.png",
    "128": "icon128.png"
  }
}
```

**Ключевые моменты:**
- `chrome_url_overrides.newtab` — указывает, что расширение заменяет новую вкладку.
- `permissions: ["storage"]` — доступ к локальному хранилищу (не требует дополнительных прав).

### 4. script.js (логика)

```javascript
// Получаем элемент textarea
const textarea = document.getElementById('notes');

// Загружаем сохранённые заметки при загрузке страницы
chrome.storage.local.get(['notes'], (result) => {
  if (result.notes) {
    textarea.value = result.notes;
  }
});

// Автосохранение при вводе с debounce (чтобы не писать часто)
let saveTimeout;
textarea.addEventListener('input', () => {
  clearTimeout(saveTimeout);
  saveTimeout = setTimeout(() => {
    const value = textarea.value;
    chrome.storage.local.set({ notes: value }, () => {
      // Опционально: показать индикатор сохранения
      console.log('Saved');
    });
  }, 500); // сохраняем через 500 мс после остановки печати
});

// Сохраняем также при закрытии/уходе со страницы (на всякий случай)
window.addEventListener('beforeunload', () => {
  chrome.storage.local.set({ notes: textarea.value });
});
```

### 5. Хранение данных

- **Место:** `chrome.storage.local` — локальное хранилище браузера, привязанное к профилю пользователя.
- **Ключ:** `notes` (строка)
- **Объём:** до 10 МБ на профиль, более чем достаточно для текстовых заметок.

### 6. Возможные расширения (опционально)

Если потребуется добавить базовое форматирование без усложнения:

- Использовать `contenteditable` вместо `textarea`, добавить кнопки для жирного/курсива.
- Сохранять форматирование в формате HTML (но это усложнит хранение и потребует санитизации).

Но MVP оставляем максимально простым — чистый текст.

## Безопасность и разрешения

- **Минимальные разрешения:** только `storage`. Никакого доступа к другим сайтам, истории, вкладкам.
- **Нет внешних запросов:** всё работает локально, не требует интернета.
- **Нет аккаунта:** пользователь не оставляет личные данные.

## Деплой и распространение

1. Упаковать папку с файлами в `.zip`.
2. Загрузить в **Chrome Web Store** (разовый платёж $5).
3. В описании указать ключевые слова: *new tab notes, simple notepad, momentum alternative, minimalist, just notes*.

## Заключение

Техническая реализация Just Notes — это классическое расширение Chrome с переопределением новой вкладки и локальным хранением. Оно занимает минимум кода, не требует серверной инфраструктуры и может быть запущено за несколько часов. Это идеальный пример **минимального жизнеспособного продукта (MVP)** с доказанным спросом.
